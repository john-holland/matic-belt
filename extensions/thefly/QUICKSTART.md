## Troubleshooting

**"Enable Microphone" doesn't work**
- Allow browser permission when prompted
- Must use HTTPS or localhost
- Try Chrome or Firefox

**Nothing appears in visualizations**
- Make sure you clicked "Start"
- For audio: Make sure microphone is enabled or play a test tone
- For galaxy: Wait a few seconds for simulation to begin

**Third space never emerges**
- Boost galaxies multiple times
- Play loud audio (integrated mode)
- Wait longer (can take 100+ simulation ticks)
- Try adding more galaxies

**Performance is slow**
- Close other browser tabs
- Use 2D mode instead of 3D for audio
- Reduce number of galaxies in galaxy sim
